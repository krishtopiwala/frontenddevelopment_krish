Math Object Practice Methods

  1. Generate a random number between 1 and 100 using Math.random().
  2. Write a program to find the square root of a number using Math.sqrt().
  3. Calculate the area of a circle given its radius using Math.PI.
  4. Round a number to the nearest integer using Math.round().
  5. Find the largest and smallest number from an array using Math.max() and Math.min().
  6. Simulate rolling a dice using Math.floor(Math.random() * 6) + 1.
  7. Calculate the power of a number using Math.pow().
  8. Round a decimal number to 2 decimal places using Math.round() and arithmetic operations.
  9. Find the absolute value of a number using Math.abs().
  10. Generate a random decimal between two numbers (e.g., 2 and 5).

String Methods Practice

  1. Count the number of vowels in a string using match() and a regular expression.
  2. Check if a string contains a specific substring using includes().
  3. Trim extra spaces from a string using trim().
  4. Extract the first 10 characters of a string using slice() or substring().
  5. Find the position of the first and last occurrence of a character in a string using indexOf() and lastIndexOf().
  6. Split a sentence into an array of words using split() and join them back using join().
  7. Check if a string starts and ends with a specific character using startsWith() and endsWith().
